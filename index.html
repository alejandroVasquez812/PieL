PieL

A programming language for Spanish speakers, designed to help those who have dificulty grasping programming or English in general. It can be considered especially helpful for children who may not understand fully by just using single keywords. It is a basic grammer consisting of variables, conditionals, and loops to give a basic understanding of skills that can translate to other programming language by defining keywords as sentences in natural language.

Java

For this project you will need to have the JDK installed and in your path. You can find the JDK here

To verify if java is installed and in your system path open a terminal and type

$ java -version

You should receive the version in the next line.

JavaCC

You will also need to install JavaCC. To download visit the JavaCC Homepage.

Building PieL

  1. Open Terminal on the folder where the .jj file is along with the javacc.jar file is.
  2. Run these commands
java -cp "path to javacc.jar" javacc "path to PieL.jj"
javac *.java

Instructions to run the language:

  1. Run a terminal inside the folder where the parser,lexer and intermidiate code reside.
java PieL
  1. This will allow you to start writing code in the PieL Language.

Example code to write (the words inside the quotations are to be replaced for information of your choice ex. ID would be the name of a variable you have created):

b) Crea un "STRUCT" de "TYPE" llamado "ID" y valores [ "MULTITERM" ].

c) Si ("STRING" OR "NUMBER" OR "ID") es igual a ("STRING" OR "NUMBER" OR "ID") {"insert instructions here"}.

d) Imprime "ID".

e) Imprime arreglo o lista "ID".
